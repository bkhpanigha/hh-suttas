Hillside Hermitage - Sutta Search Panel


1. Language Selection: Choose your search language. Note that some Pāli terms may appear in translations and comments.

2. Book Selection: Define which books you want to search through.

3. Exact Match: Enable exact match to limit results to precise terms only. For example, if you search for "training", you'll only see results that contain that exact word, not words like "restraining".

4. Single Result: Limit display to first occurrence per sutta.

5. Comments Only: Restrict search to comments only.

6. Search: Enter your terms and press Enter.


Dark Mode
Enable dark mode using the button at the top of the page.

Troubleshooting
If you encounter an issue with the search, try re-enabling the "Use offline" setting. If the error persists, you may want to report it via the "Give feedback" button.
